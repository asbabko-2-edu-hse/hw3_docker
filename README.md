# Домашнее задание 3: Docker

## Описание проекта
Проект состоит из генератора и аналитика данных (CSV-файла с данными о студентах, организующих мероприятия)

## Команды для запуска
### Windows:
- `run.bat build_generator` - сборка образа генератора
- `run.bat run_generator` - генерация data.csv
- `run.bat build_reporter` - сборка образа аналитика  
- `run.bat run_reporter` - генерация report.html
- `run.bat clear_data` - очистка папки data

### Mac/Linux:
- `run.sh build_generator` - сборка образа генератора
- `run.sh run_generator` - генерация data.csv
- `run.sh build_reporter` - сборка образа аналитика  
- `run.sh run_reporter` - генерация report.html
- `run.sh clear_data` - очистка папки data
